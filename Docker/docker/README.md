# docker

#### 项目介绍
centos php mysql tomcat nginx容器镜像dockerfile文件

#### 安装教程

1. yum -y install docker
2. systemctl start docker
3. systemctl enable docker

#### 使用说明

1. git clone 
2. docker build 
3. docker run
